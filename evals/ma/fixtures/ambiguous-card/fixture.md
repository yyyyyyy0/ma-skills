# Ambiguous card fixture

- Product: Team collaboration SaaS
- Element: task card in a kanban board
- Current problem: card contains title, assignee avatar, due date, priority badge, tag list, and a truncated description — all at equal visual weight
- Card contents:
  - Task title (max 80 chars)
  - Assignee avatar (24px circle)
  - Due date label
  - Priority badge (low / medium / high / urgent)
  - Tags (1–4 tags)
  - Description snippet (2 lines, truncated)
- Context:
  - Cards are displayed in columns of 280px width
  - Users scan columns to find overdue or high-priority items
  - Clicking a card opens a detail panel
